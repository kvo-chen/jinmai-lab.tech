# 增加文化主题活动日历的活动数量

## 目标
将文化主题活动日历中的活动数量从当前的12个增加到30个左右，丰富活动内容和类型分布。

## 实现方案
1. **修改文件**：`src/services/eventCalendarService.ts`
2. **在events数组中添加18个左右的新活动**
3. **确保活动类型分布均匀**：
   - 主题活动（theme）
   - 协作活动（collaboration）
   - 竞赛（competition）
   - 工作坊（workshop）
   - 展览（exhibition）
4. **确保活动状态分布合理**：
   - 即将开始（upcoming）
   - 进行中（ongoing）
   - 已结束（completed）
5. **活动内容多样化**：涵盖不同的文化元素、主题和地区
6. **每个活动的字段都符合CulturalEvent接口要求**

## 新增活动规划
- 新增6个主题活动
- 新增4个协作活动
- 新增4个竞赛活动
- 新增3个工作坊活动
- 新增1个展览活动

## 实施步骤
1. 打开`eventCalendarService.ts`文件
2. 在events数组末尾添加新的活动对象
3. 确保每个新活动都有唯一的id、完整的字段和合理的数据
4. 保存文件
5. 验证活动列表在前端正确显示

## 预期效果
- 活动日历中显示30个左右的活动
- 活动类型和状态分布均匀
- 活动内容多样化，涵盖不同文化元素
- 前端展示正常，筛选和搜索功能不受影响
