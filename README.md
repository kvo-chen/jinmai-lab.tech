# AI共创平台

基于AICE互动-转化闭环框架的AI驱动型用户共创平台，赋能老字号品牌年轻化转型。

## 平台简介

AI共创平台旨在实现用户从吸引、共创、展示到战略采纳的全流程管理，通过AI技术助力传统文化的创新表达和商业化落地。

## 核心功能

### 用户端功能
- **注册与登录系统**：支持邮箱登录，用户画像标签体系
- **AI创作工具集**：
  - 低门槛设计工具（一键国潮设计、AI滤镜）
  - 文化资产嵌入功能（老字号专属素材库）
  - 实时文化溯源提示（创作过程中展示文化元素来源）
- **作品管理中心**：
  - 作品编辑/保存/发布功能
  - 草稿箱与版本历史
- **社区互动系统**：
  - 作品展示与点赞评论
  - 排行榜与荣誉标签体系
  - 创作者交流社区

### 管理端功能
- **内容审核系统**：作品合规性与文化准确性审核
- **数据分析面板**：
  - 用户行为数据可视化
  - 作品传播效果分析
  - 文化植入效果评估
- **战略采纳管理**：
  - 优秀作品评选流程
  - 品牌商业化应用对接
  - 青年创意官认证管理

## 技术架构

- **前端**：React 18+、TypeScript、Tailwind CSS、Framer Motion、Recharts
- **构建工具**：Vite
- **数据存储**：LocalStorage（模拟）
- **响应式设计**：支持多终端访问

## 安装与运行

1. 克隆项目代码
```bash
git clone <项目地址>
```

2. 安装依赖
```bash
npm install
# 或
pnpm install
```

3. 启动开发服务器
```bash
npm run dev
# 或
pnpm dev
```

4. 构建生产版本
```bash
npm run build
# 或
pnpm build
```

## 使用说明

### 普通用户
- 注册账号并登录
- 浏览探索页面查看其他创作者的作品
- 进入创作中心使用AI工具进行创作
- 在个人控制台管理自己的作品和查看数据统计

### 管理员
- 使用管理员账号登录
- 进入管理控制台查看平台数据概览
- 审核用户提交的作品
- 管理商业化申请和用户账户

## 部署说明

项目构建完成后，可以将`dist`目录下的文件部署到任何静态文件服务器上。推荐的部署方式：

1. **Vercel**：适合个人和小团队快速部署
2. **Netlify**：提供持续部署和CDN加速
3. **GitHub Pages**：适合开源项目
4. **自建服务器**：使用Nginx、Apache等Web服务器

## 注意事项

1. 当前版本使用LocalStorage模拟数据存储，实际部署时应替换为后端API
2. 项目支持明暗主题切换，默认跟随系统设置
3. 所有AI生成的内容使用模拟数据，实际应用中应集成真实的AI服务
4. 管理端功能需要管理员权限才能访问
